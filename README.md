TimeTracker

Tämän projektin tarkoituksena oli harjoitella responsiivisen käyttöliittymän rakentamista. Sovellus näyttää eri tehtävien ajankäytön kortteina ja sisältää profiilikortin, josta voi vaihtaa ajanjaksoa (Daily / Weekly / Monthly). Projektin idea ja sisältö (ml. kuvat, valmis data) on lähtöisin Frontend Mentor -sivustolta.




**Teknologiat**
- React (JSX)
- Vite
- Tailwind CSS

**Käyttöönotto (paikallisesti)**
1. Kloonaa repositorio:

```bash
git clone https://github.com/AlinaLokkinen/timetracker.git
```

2. Asenna:

```bash
npm install
```

3. Käynnistä kehityspalvelin:

```bash
npm run dev
```

Tutustu sovellukseen lokaalisti http://localhost:5173.
